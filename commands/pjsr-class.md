# PJSR Class Documentation

Look up detailed documentation for a PJSR class. Use the `pjsr_class_info` tool with the class name.

## Common Classes to Look Up

**Image Processing:**
- `Image` - Core 2D image manipulation
- `ImageWindow` - Image window interface
- `View` - View interface (main or preview)
- `ProcessInstance` - Execute PixInsight processes

**UI Components:**
- `Dialog` - Modal dialog windows
- `Control` - Base UI control class
- `Sizer` - Layout management
- `Label`, `PushButton`, `CheckBox`, `Edit`
- `ComboBox`, `SpinBox`, `Slider`
- `TreeBox`, `ViewList`, `GroupBox`

**Graphics:**
- `Bitmap`, `Graphics`, `Pen`, `Brush`, `Font`

**Data:**
- `Matrix`, `Vector`, `Point`, `Rect`
- `File`, `ByteArray`

If the user asks about a class, use `pjsr_class_info` to get full documentation including constructors, properties, methods, and event handlers.
